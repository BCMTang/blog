# 浏览器尺寸

> 屏幕大小、浏览器类型、设置都有影响。

```js
document.documentElement.clientHeight
```

> 屏幕大小：1920*1080、默认非全屏`F11`、100%比例。不同浏览器可视高度（px）：

|           | 显示书签 | 隐藏书签 |
|-----------|------|------|
| 谷歌Chrome  | 923  | 955  |
| 火狐Firefox | 919  | 947  |
| 微软Edge    | 929  | 961  |

> 另外，如果有顶条的话，默认55px
