## 安装

- 官网选择版本安装
- 低版本手动配置环境变量

```shell
node -v
```

> 全局依赖默认地址：`C:\Users\Admin\AppData\Roaming\npm\node_modules`

```shell
npm i -g [依赖名]
```

## 卸载

- 电脑系统里卸载`Node.js`
- 删除文件夹：`C:\Users\Admin\AppData\Roaming\npm`、`C:\Users\Admin\AppData\Roaming\npm-cache`
